Apa itu Html?

Html atau Hypertext Markup Leanguage adalah sebuah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan di peramban (browser) internet. Program ini dapat dibantu dengan teknologi Cascading Style Sheets (CSS) dan bahasa scripting dengan JavaScript dan VBSscript. Elemen HTML biasanya memiliki tag pembuka dan penutup yang mengelilingi dan memberi makna pada konten. Misalnya, elemen yang berbeda dapat mendeskripsikan teks sebagai judul, paragraph atau item daftar.
