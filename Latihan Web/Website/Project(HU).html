Apa itu Programming?

Programming adalah sebuah proses membuat sebuah program di komputer, bisa berupa software, aplikasi android, website dan masih banyak lagi. Banyak sekali bahasa pemrogramman yang ada saat ini, antara lain Html, Js, Python, C#, Php dan masih banyak lagi.

Banyak yang merngira bahwa semua program yang ada pada komputer itu sulit. pendapat itu tidak sepenuhnya benar dan juga tidak salah. Contohnya seperti pemrograman web yang tidak terlalu memerlukan algoritma pada penerapannya, seperti dalam Html dan CSS. Alasan kenapa program web adalah salah satu program yang hanya harus memahami setiap elemen nya saja dan dengan sedikit sintaks rumit. bagi kalian yang baru ingin masuk ke dunia programming, program web adalah salah satu opsi terbaik.
